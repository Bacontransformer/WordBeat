# Offline data (Android / Capacitor)

Generated by `npm run data:offline` from `server/data/raw/*.json` and the same level seeds as `initDb.ts`.

- `packs.json` — junior / cet4 / senior word lists
- `levels.json` — six levels (path, waves, modules, textbook HP); words sampled at runtime

Do not edit by hand unless you know what you are doing; re-run the export script after changing seeds or vocab.
