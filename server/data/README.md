# 下载开源词库到 server/data/raw

来源：https://github.com/KyleBing/english-vocabulary

```powershell
curl.exe -sL -o server/data/raw/junior.json "https://raw.githubusercontent.com/KyleBing/english-vocabulary/master/json/1-%E5%88%9D%E4%B8%AD-%E9%A1%BA%E5%BA%8F.json"
curl.exe -sL -o server/data/raw/cet4.json "https://raw.githubusercontent.com/KyleBing/english-vocabulary/master/json/3-CET4-%E9%A1%BA%E5%BA%8F.json"
curl.exe -sL -o server/data/raw/senior.json "https://raw.githubusercontent.com/KyleBing/english-vocabulary/master/json/2-%E9%AB%98%E4%B8%AD-%E9%A1%BA%E5%BA%8F.json"
```
